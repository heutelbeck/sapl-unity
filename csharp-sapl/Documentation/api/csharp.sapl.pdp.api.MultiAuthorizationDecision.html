Class MultiAuthorizationDecision

A multi-decision holds a map of authorization subscription IDs and corresponding authorization decisions. It provides methods to add single authorization decisions related to an authorization subscriptionID, to get a single authorization decision for a given authorization subscription ID and to System.Collections.IEnumerator over all the authorization decisions.
